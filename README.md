# velox

Thin web-ui starter.

<br />




## bare -> base | bare-styled -> base-styled

* typescript
* babel
* eslint
* webpack
* webpack-dev-server
* base libs:
    - [@xcmats/js-toolbox](https://drmats.github.io/js-toolbox/)
    - [mem-box](https://drmats.github.io/mem-box/)
* simple `css` + `favicon`

<br />




## development

```
npm i
npm run lint
npm run dev
```

<br />




## production build

```
npm run build
```
