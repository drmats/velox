## base-react-baseui-styled -> base-react-baseui-maplibre

* typescript
* babel
* eslint
* webpack
* webpack-dev-server
* simple `css` + `favicon`
* react
* base libs:
    - [@xcmats/js-toolbox](https://drmats.github.io/js-toolbox/)
    - immer
    - [mem-box](https://drmats.github.io/mem-box/)
    - prop-types
    - react-redux
    - [red-g](https://drmats.github.io/red-g/)
    - redux
    - redux-thunk
    - lodash.throttle
* ui:
    - baseweb.design
* maplibre-gl
* react-map-gl
