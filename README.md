# velox

Thin web-ui starter.

<br />




## base-react | bare-styled -> base-react-baseui-styled

* typescript
* babel
* eslint
* webpack
* webpack-dev-server
* simple `css` + `favicon`
* react
* base libs:
    - [@xcmats/js-toolbox](https://drmats.github.io/js-toolbox/)
    - immer
    - [mem-box](https://drmats.github.io/mem-box/)
    - prop-types
    - react-redux
    - [red-g](https://drmats.github.io/red-g/)
    - redux
    - redux-thunk
* ui:
    - baseweb.design

<br />




## development

```
npm i
npm run lint
npm run dev
```

<br />




## production build

```
npm run build
```
